# HTB - Meow

## Цель

Получить первый флаг на платформе Hack The Box.

## Что изучила

### OpenVPN

* Установила и настроила OpenVPN.
* Подключилась к сети Hack The Box.

### Nmap

Использовала команды:

nmap --version

nmap scanme.nmap.org

nmap -Pn -p 23 <IP>

Изучила:

* что такое порты;
* что такое сервисы;
* как находить открытые порты.

### Telnet

Узнала, что порт 23 используется сервисом Telnet.

Подключение:

telnet <IP>

### Linux

Использовала команды:

ls

cat flag.txt

Изучила:

* просмотр файлов;
* чтение содержимого файлов;
* работу с домашней директорией пользователя root.

## Результат

Успешно получила первый флаг на Hack The Box.

## Полученные навыки

* OpenVPN
* Nmap
* Telnet
* Linux
* Работа с удалёнными системами
